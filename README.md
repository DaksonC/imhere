# imhere
App que adiciona participantes a um evento.
Projeto do curso ignite da Rocketseat Fundamentos de React Native e TypeScript.

## Instalação e execução do projeto

-          git clone https://github.com/DaksonC/imhere.git
-          cd imhere
-          yarn
-          yarn start

## tecnologias

- React Native
- TypeScript
